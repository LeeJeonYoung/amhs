# 02. 배선표 (로봇 / 허브)

> ⚠️ 핀 번호는 **에듀이노 키트 기본 배선 기준 가정값**입니다.
> 키트 설명서를 보고 모터/센서가 실제로 쓰는 핀과 **대조**한 뒤,
> 겹치면 `robot.ino` 상단 `#define` 만 고치면 됩니다.
> nRF24 가 쓰는 **9, 10, 11, 12, 13** 핀은 비워둬야 합니다.

## 로봇 (각 3대 동일)

### nRF24L01 (어댑터 경유)
| 어댑터 핀 | 아두이노 | 비고 |
|---|---|---|
| VCC | **5V** | 어댑터(AMS1117)가 3.3V로 변환 |
| GND | GND | |
| CE  | D9  | |
| CSN | D10 | |
| SCK | D13 | SPI 고정 |
| MOSI| D11 | SPI 고정 |
| MISO| D12 | SPI 고정 |
| IRQ | 연결 안 함 | |

> nRF24 모듈(수핀)은 어댑터 위 소켓에 직접 꽂고, **어댑터 → 아두이노**는 암–수 점퍼선 7가닥.

### 모터드라이버 (L298N)
| L298N | 아두이노 |
|---|---|
| ENA | D5 (PWM) |
| IN1 | D2 |
| IN2 | D4 |
| IN3 | D7 |
| IN4 | D8 |
| ENB | D6 (PWM) |

### IR 라인센서 2개
| 센서 | 아두이노 |
|---|---|
| 좌(LEFT)  | A0 |
| 우(RIGHT) | A1 |

### 초음파 HC-SR04
| HC-SR04 | 아두이노 |
|---|---|
| Trig | A2 |
| Echo | A3 |
| VCC/GND | 5V / GND |

## 허브 (1대, 모터·센서 없음)
nRF24L01 **딱 하나만** 위 "nRF24L01" 표와 똑같이 연결.
그리고 허브 아두이노는 **USB 케이블로 Mac에 연결**.

```
[Mac] ──USB── [허브 UNO + nRF24] ))) 무선 ((( [로봇1/2/3 UNO + nRF24]
```

## 핀 충돌 점검 체크
- [ ] 키트 모터/센서가 D9~D13 을 쓰지 않는가? (쓰면 다른 핀으로 이동)
- [ ] nRF24 VCC 는 5V(어댑터)에 꽂았는가? (3.3V 아님 — 어댑터가 변환)
- [ ] 모든 보드 GND 공통 연결?
