# 03. 트랙(라인) 제작

## 핵심 원칙
그냥 원형 트랙이면 로봇이 줄줄이 따라가기만 해서 **알고리즘이 필요 없음**.
**교차로(충돌 지점)** 를 만들어야 교통관리·구역락·교착(deadlock)을 테스트할 수 있다.

## 추천: 8자 트랙 (교차로 1개)
```
   ╭──────╮
   │      │
    ╲    ╱
     ╲  ╱
      ╳   ← 교차로 (양쪽 IR센서가 동시에 검정 = "노드 감지")
     ╱  ╲
    ╱    ╲
   │      │
   ╰──────╯
```
- 로봇들이 계속 순환 → 교차로에서 자연스럽게 경쟁 발생
- 펌웨어/서버가 이 교차로를 **구역 락**으로 관리 (한 번에 한 대만 통과)

## 제작 팁
- **바닥**: 흰색/밝은색 (흰 장판·폼보드·전지 이어붙이기). 대비가 핵심.
- **테이프**: 검정 절연테이프, 폭 **약 2cm**.
- **곡선**: 센서 2개라 급커브에 약함 → **완만하게 둥글게** (직각 X).
- **크기**: 1.5×1.5m ~ 2×2m 면 충분.
- **교차로**: + 자로 두 선이 직교하게. 통과 시 양쪽 센서가 동시에 검정을 읽어야 함.

## 동작 원리 (센서 2개 기준)
| 좌센서 | 우센서 | 의미 | 로봇 동작 |
|---|---|---|---|
| 흰 | 흰 | 라인이 두 센서 사이(중앙) | 직진 |
| 검정 | 흰 | 좌로 라인 → 우로 치우침 | 좌회전 |
| 흰 | 검정 | 우로 라인 | 우회전 |
| 검정 | 검정 | **교차로** | 정지 → 서버 허가 대기 |

## 확장: 격자(田) 트랙
교차로 여러 개 → 경로 선택(라우팅) + 다중 교착까지. 8자에서 익숙해지면 시도.
```
N1─N2─N3
│  │  │
N4─N5─N6
│  │  │
N7─N8─N9
```

> 트랙은 테이프라 언제든 떼었다 다시 붙일 수 있음. **8자부터 시작**.
