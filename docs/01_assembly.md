# 01. 조립 가이드

## 준비물 점검 (도착 확인)
- [ ] 에듀이노 라인트레이서 키트 × 3
- [ ] nRF24L01 모듈 × 4~6 (로봇3 + 허브1 + 예비)
- [ ] AMS1117 어댑터 × 4~6
- [ ] 허브용 아두이노 우노 호환보드 × 1 (+USB 케이블)
- [ ] 암–수 듀폰 점퍼선 (40핀)
- [ ] AA 건전지 (4개 × 3대 = 12개+)
- [ ] 검정 절연테이프 + 밝은 바닥

## 1단계: 키트 3대 조립 (설명서대로)
에듀이노 키트의 조립서/영상대로 차체·모터·바퀴·센서·UNO 보드를 조립.
**라인트레이싱 기본 동작까지 키트 예제로 1대 먼저 확인**하면 이후 디버깅이 쉬움.

## 2단계: nRF24 모듈 ↔ 어댑터
- nRF24 모듈의 8핀(수)을 **어댑터 위 소켓에 그대로 꽂기** (납땜 없음).
- 방향 주의: 핀 라벨(VCC/GND/CE…)이 어댑터와 맞도록.

## 3단계: 어댑터 ↔ 아두이노 (암–수 점퍼 7가닥)
`docs/02_wiring.md` 표대로 7가닥 연결. 로봇 3대 + 허브 1대 = 동일 배선 4세트.
- VCC → **5V** (3.3V 아님! 어댑터가 변환)
- GND → GND
- CE→9, CSN→10, SCK→13, MOSI→11, MISO→12

## 4단계: 허브 만들기
허브용 UNO 1대에 nRF24(어댑터 포함)만 연결 → **USB로 Mac에 꽂기**.
허브는 모터·센서 없음. 무선↔USB 중계기.

## 5단계: 배선 사진 찍어두기
3대가 똑같아야 하므로, 1대를 깔끔히 배선한 뒤 **사진**을 찍어 나머지 2대를 똑같이.
